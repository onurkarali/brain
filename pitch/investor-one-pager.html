Brain Memory

Give AI coding assistants a long-term memory
Open Source · MIT npm: brain-memory
Brain Memory is an open-source plugin that gives AI coding assistants persistent, human-like memory across sessions — so they remember your projects, decisions, and preferences forever.

The Problem

  • AI coding assistants forget everything when you start a new session
  • Developers waste time re-explaining context, past decisions, and preferences
  • No existing solution is local, transparent, or truly intelligent about what to remember

The Opportunity

  • AI-assisted coding is a $40B+ market growing 25%+ annually
  • Memory is the #1 requested feature by power users of Claude, Copilot, and Cursor
  • First mover in the "AI agent memory layer" category
What We Do
💾

Remember

The AI stores decisions, learnings, and context as simple Markdown files organized in a meaningful folder structure.

🔗

Connect

Memories automatically link together. Recalling one idea surfaces related knowledge — just like the human brain.

🌙

Evolve

A "sleep cycle" strengthens important memories, merges duplicates, and lets irrelevant ones fade — keeping the brain sharp.
